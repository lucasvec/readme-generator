# Repositório

## .idea
Pasta contendo arquivos de configuração do ambiente de desenvolvimento IntelliJ IDEA.
- `inspectionProfiles:`: Subpasta contendo perfis de inspeção de código.
  - `profiles_settings.xml`: Arquivo de configurações dos perfis de inspeção.
- `.gitignore`: Arquivo que especifica arquivos e pastas ignorados pelo Git.
- `misc.xml`: Arquivo de configurações diversas do projeto.
- `modules.xml`: Arquivo que define os módulos do projeto.
- `readme-generator.iml`: Arquivo de módulo do IntelliJ IDEA.
- `vcs.xml`: Arquivo de configurações do sistema de controle de versão.

## ia_prompt
Pasta contendo os arquivos relacionados à interação e processamento de prompts de IA.
- `__pycache__:`: Subpasta contendo arquivos bytecode do Python.
  - `code_template.cpython-312.pyc`: Arquivo bytecode de `code_template.py`.
  - `reader.cpython-312.pyc`: Arquivo bytecode de `reader.py`.
- `code_template.py`: Arquivo contendo um modelo de código.
- `ia_request.py`: Arquivo responsável por fazer requisições à IA.
- `reader.py`: Arquivo para leitura de dados.
- `test.ipynb`: Notebook Jupyter para testes.

- `.gitignore`: Arquivo que especifica arquivos e pastas ignorados pelo Git.
- `LICENSE`: Arquivo contendo a licença do projeto.
- `README.md`: Arquivo contendo a documentação do projeto.
- `update.py`: Arquivo para realizar atualizações.
